## Library Management

- Import EPUB, MOBI, AZW, AZW3, and PDF files
- View, sort, search, and filter your library
- Mark books as read/unread
- Bulk-import from an existing Calibre library

## Metadata

- Reads embedded metadata from all supported formats
- Edit metadata manually (title, author, series, etc.)
- Fetch metadata from Open Library and Google Books

## Device Support

- Automatic USB device detection
- Send books directly to a connected Kindle
- Linux: full MTP support
- Windows: removable-drive detection
