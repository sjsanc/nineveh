# Nineveh

Nineveh is a minimal Calibre replacement. It manages your library of ebooks and integrates with ereaders, aiming to replicate Calibre's most useful features.

It's built using Claude, Wails 2 and React, and very much a WIP.

## Features

- View/sort/search/filter your ebook library
- Add/remove ebooks
- Mark books as read
- Supports PDF, EPUB, AWZ, AWZ3 and MOBI
- Import from Calibre
- Device detection (currently just Kindle on Linux)
- Send to device
- Fetch/edit metadata
